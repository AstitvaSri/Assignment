1 Objective of Lab

The objective of this lab is to get familiar with Javascript and its capabilities.

  • We will recall HTML and CSS.
  • We will learn how to run Javascript inside and outside the browser
    • type C-Shift-k in Firefox, C-Shift-j in Chrome.
    • How to write inline script using <script>.
    • How to include external JS file using <script>.
  • We will learn about module system in Javascript and also look at node, npm and webpack.
  • We will learn about Strings and Arrays and their commonly used methods like
    • str.charAt/str.charCodeAt
    • str.indexOf/lastIndexOf
    • str.replace
    • str.substring/str.substr
    • str.toLowerCase/str.toUpperCase
    • str.trim
    • str.split
    • arr.push/arr.pop/arr.shift/arr.unshift
    • arr.concat
    • arr.sort
    • arr.slice/arr.splice
    • arr.indexOf
    • arr.join
  • We will look at the Document Object Model (DOM) of the browser.
    • DOM elements
    • Methods to retrieve elements: getElementById, getElementsByClassName, etc.
    • Manipulating HTML/CSS using JS: (element.innerHTML, element.style).
    • The window and document objects.
  • We will learn about event handling:
    • <button onclick = "alert('Hello!')">
    • document.getElementById(...).onclick = function()
    • element.addEventListener(...)
    • Defining handlers for the following events: click, hover, change, onkeyup/down.

3 Problems

3.1 Grade students

This problem is in four parts:

3.1.1 Fetch the details

  1. Create files named details.js and test-details.js.
  2. Install a npm package called itws2-sample-marks-data and include it in your test-details.js
  3. This included module is a JSON object which contains marks (out of 100) for the students in the following format: <name>: [marks1, marks2, marks3, marks4, marks5].

  4. Now write a function (with name as specified) for each of the following in your details.js and export them all as well.

    • a. getHighestMarks : This function should take the data

    (as mentioned above) and return name of the student with highest total.

  • b. getSubject2Toppers : This function should take the data

and return a sorted array of students' name with highest marks2.

  • c. getSubjectAverage : This function takes the data and

and returns array of length 5, which contains average marks for each subjects respectively.

  • d. getAbove : This function takes 2 arguments. First is data

and the other is a number between 0 to 100. It should return sorted array of students' name which have percentage above the number given in the second parameter.

  1. Now include all the above written functions in your test-details.js file and print (console.log) the output for each. You should pass correct parameters for each of the functions. (For forth part, give 85 as second argument).

3.1.2 Web Interface

Open a new html file named details.html

  • Using the data from the previous section, perform the following tasks:
    • Create a heading (<h1>) named Student Sheet - in the <body> of html page.

      • Create an HTML table(<table>) with the following header(<thead>).

        Name marks1 marks2 marks3 marks4 marks5
  • Populate the table with entries from the JSON object.
  • Highlight the row with color green, on hover over any row of the table.

3.1.3 Dynamic Webpage

Open a new HTML file named dynamic.html and perform the following three tasks:

Dropdown
Create a dropdown(<select>) of names of the students.
Displaying marks
On selecting any student name, display the corresponding attributes of the selected student in a table with the following format:
      | Name | marks1 | marks2 | marks3 | marks4 | marks5 | Percentage | Grade |

Criteria for grade:

%age Grade
>90 A
80-90 B
70-80 C
60-70 D
<60 F
Highlight
Also, highlight the minimum and the maximum marks with red and green respectively.

3.1.4 BONUS

  • Open a HTML file named highlight.html.
  • Using the table created in the Web Interface subproblem, implement the following feature.
    • Highlighting the cell of the table on mouse click.
    • Moving the highlight on pressing arrow keys.

3 Problems

3.1 Person Details

3.1.1 Output the details

  • Open a new file named persondetails.txt in your terminal.
  • Open console in your browser and define a class Person
  • Person class should contain the following fields
    • Name
    • Age
    • Sex
  • Initialize an object person of class Person with values for name, age and, sex. Print (console.log) the person object and copy the output to persondetails.txt file.
  • Create a member function for the class, getAge(). This function should return the person's age.
  • Call the person.getAge() function and copy the output to persondetails.txt file.
  • Define a new function getName() to the class prototype by using Person.prototype.getName = function() { ... }. This function should return the name of the person.
  • Call the person.getName() function and copy the output to persondetails.txt file.
  • Finally, copy the code for the class and the prototype function definition to personcode.js file.

3.1.2 Classroom Details

  • Open a new file named classroomdetails.txt in your terminal.
  • Open console in your browser and define a class Teacher. This class should inherit the Person class and have the following additional fields
    • Subject
  • Define another class Student. This class should inherit the Person class and have the following additional fields
    • Roll Number
    • Subjects
  • Copy all the lines from file persons.txt and paste it in the console.

  • Now, create a function getParticipants(subjectName, persons). This function takes two parameters: the subject name and a list of persons(including teachers and students). The function should return details in the following manner:

    Teacher:

    <name>

    Students:

    <rollNo1>

    <rollNo2>

  • Call the getParticipants function with parameters: ("itws", people). Print the output(console.log) of the function and copy it to classroomdetails.txt.
  • Finally, copy the source code for the classes to classroomcode.js file.
